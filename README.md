# Swati Portfolio Website

A modern responsive personal portfolio website for a Junior Data Analyst built using pure Vanilla HTML, CSS, and JavaScript.

This project was inspired by a custom UI/UX portfolio design and developed without any frontend frameworks or backend technologies.

---

# Preview

This portfolio includes:

- Hero section
- About section
- Tech stack section
- Projects showcase
- Footer
- Responsive design

---

# Tech Used

- HTML5
- CSS3
- Vanilla JavaScript

No React.
No Node.js.
No frameworks.
